## Customization

### Colors
The website uses a blue color scheme. You can customize the colors by modifying the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #1d4ed8;
    --accent-color: #10b981;
}
```
